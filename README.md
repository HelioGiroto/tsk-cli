# tsk-cli

Treasury of Scripture Knowledge Command Line Interface

It's dead simple. Running this:

    python tsk-cli.py jo3:16

Outputs this:

    # John 3:16
    
    ## God
    
    Luke 2:14
    Romans 5:8
    2 Corinthians 5:19
    Titus 3:4
    1 John 4:9
    
    ## was
    
    1 John 3:16
    John 3:16
    Romans 5:8
    Romans 8:32

## Dependencies

- bible 0.2
